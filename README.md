# Retail Sales GenAI Assistant

An AI-powered Retail Sales Analytics Assistant that enables users to analyze retail sales data using natural language.

The project combines Google Gemini 2.5 Flash, SQLite, Pandas, Plotly, and Gradio to generate SQL queries, analyze business data, and provide actionable insights.

---

## Features

- Natural language retail analytics
- AI-generated SQL queries
- Business insight generation
- SQLite database integration
- Interactive chatbot interface
- Monthly sales trend analysis
- Product performance analysis
- Promotional sales analysis
- Chat report export

---

## Tech Stack

- Python
- Google Gemini 2.5 Flash
- SQLite
- Pandas
- Plotly
- Gradio
